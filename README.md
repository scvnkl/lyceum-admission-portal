# Lyceum Admission Portal

A digital portal for tracking admission to the Infotech Lyceum.

## Features

- applicant registration
- personal dashboard
- admission progress tracking
- exam results
- communication with admission office

## Project structure

frontend - website and applicant portal  
backend - API and business logic  
database - database schema  
docs - product documentation

## Team

Project created by students of the Infotech Lyceum
